# Modelo de Machine Learning para Previsão de Crédito

Projeto desenvolvido em Python para prever o score de crédito de clientes com base em dados históricos.

## Objetivo
Criar um modelo de Machine Learning capaz de analisar características dos clientes e prever a probabilidade de aprovação de crédito.

## Tecnologias utilizadas
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## Algoritmos utilizados
- Random Forest
- K-Nearest Neighbors (KNN)

## Etapas do projeto
1. Importação da base de dados
2. Tratamento e limpeza dos dados
3. Análise exploratória
4. Treinamento dos modelos
5. Avaliação da acurácia
6. Geração de previsões

## Arquivos do projeto

- `modelo_credito.ipynb` → Notebook com todo o desenvolvimento do modelo
- `clientes.csv` → Base de dados utilizada para treinamento e teste
